# Versão 1 (arquivada) — ChefBot original

Este diretório contém a versão inicial do ChefBot, entregue no **primeiro
trabalho** da disciplina. Foi preservada aqui para referência histórica.

Arquitetura: NLTK + Naive Bayes + Flask, **sem LLM**.

A versão atual (raiz do repositório) é o **Trabalho Final**, com
arquitetura híbrida (base estruturada + LLM Qwen2.5-7B como fallback).
Ver [../README.md](../README.md) e [../TRABALHO_FINAL.md](../TRABALHO_FINAL.md).
